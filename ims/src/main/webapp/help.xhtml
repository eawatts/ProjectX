Help

Notifications
Why are there yellow symbols at the top?

This shows there are warnings to do with this area that need to be looked at and are usually important

How do I stop the symbols being yellow?

Hover over your chosen symbol and then click the x button next to the notification once it has been read.

Purchase Orders
How to create Purchase Order?

Either auto generate a single purchase order with the top 25 products or create a personal purchase order using the create purchase order page.

How do I view the products that were ordered?

Click the expand button on the row you want to view.

Products
How to put a product as inactive?

Click the big button next to the product order and change the status to active.

How do I edit the product details?

Click the edit button on the row you want to change. then you change the item you want to change by clicking the individual edit button before clicking the save button to edit the information.

Suppliers
How to add a supplier?

Click the add button fill in the popup and click the submit button to add a new supplier to the system.

How do I edit the supplier details?

Click the edit button on the row you want to change. then you change the item you want to change by clicking the individual edit button before clicking the save button to edit the information.
